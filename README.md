# Netflix-EDA-movies-analysis-using-python-libraries

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Netflix Movies and TV Shows dataset using Python. The objective is to clean the dataset, analyze trends, and extract meaningful insights through visualizations.

The analysis focuses on understanding Netflix's content library based on content type, release year, genres, ratings, countries, and duration.

---

## 🎯 Objectives

- Clean and preprocess the dataset.
- Handle missing values and duplicate records.
- Analyze the distribution of Movies and TV Shows.
- Identify the most common genres and ratings.
- Study trends in content released over the years.
- Visualize important insights using charts.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
